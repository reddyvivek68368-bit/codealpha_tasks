🌸 **Project Completed: Iris Flower Classification | CodeAlpha Data Science Internship — Task 1**

Excited to share my first project from the CodeAlpha Data Science
Internship — a complete, end-to-end Machine Learning system that classifies
Iris flowers into Setosa, Versicolor, or Virginica based on their sepal and
petal measurements.

🔧 **What I built:**
✅ Full data cleaning pipeline (missing values, duplicates, IQR-based outlier handling)
✅ In-depth Exploratory Data Analysis with 10+ visualizations
✅ Feature engineering (area & aspect-ratio features)
✅ Trained & compared 7 ML models — Logistic Regression, KNN, SVM, Decision
   Tree, Random Forest, Naive Bayes, and Gradient Boosting
✅ Selected the best model (Support Vector Machine — 96.7% test accuracy)
   using cross-validation and multiple metrics (precision, recall, F1)
✅ Built an interactive **Streamlit dashboard** with live filters, KPIs,
   and a real-time prediction tool

📊 **Key takeaway:** Petal measurements alone almost perfectly separate the
three species — a great reminder that good EDA often tells you which
features matter before you even train a model.

🛠️ **Tech stack:** Python, pandas, scikit-learn, matplotlib, seaborn,
Plotly, Streamlit
